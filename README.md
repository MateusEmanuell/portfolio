# Portfólio Profissional | Mateus Emanuel

Repositório contendo os arquivos do meu portfólio pessoal de desenvolvimento.

O site está publicado e disponível em: **[mateusemanuel.com.br](https://mateusemanuel.com.br)**

## Tecnologias e Recursos
- HTML5 semântico, CSS3 (responsivo e variables-based) e JavaScript puro (Vanilla).
- Painel de acessibilidade integrado (alto contraste, fonte para dislexia e ajustes de fonte).
- Formulário de contato assíncrono conectado via Webhook.
- Hospedagem no GitHub Pages com domínio personalizado e DNS gerenciado.

## Como Executar Localmente
1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador de preferência.
